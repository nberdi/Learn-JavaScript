What is JavaScript?

  • JavaScript is a programming language that adds interactivity to websites.
  • It makes web pages dynamic by responding to user actions like clicks and typing.
  • You can use JavaScript to create animations, forms, and interactive elements.
  • It runs in the browser, which means users don't need to install anything.
  • JavaScript works with HTML and CSS to build modern websites and web apps.
  • Popular websites use JavaScript to improve user experience and functionality.

Getting Started with JavaScript

  1. Create an HTML File: Open a text editor and save the file as index.html.
  2. Add Basic HTML: Add the basic structure with <!DOCTYPE html>, <html>, <head>, and <body>.
  3. Add a Script Tag: Inside the <body>, add a <script> tag.
  4. Insert JavaScript Code: JavaScript code is inserted between the <script> and </script> tags in HTML.
  5. Write Your First Script: Add a simple alert like alert('Hello, JavaScript!');.
  6. Save and Open: Save the file and open it in your browser.
  7. Explore the Browser Console: Right-click and select "Inspect" to open the console and test JavaScript.

Understanding Variables in JavaScript

Think of a variable like a labeled box where you can store information. Just like you put your favorite snacks in a box and label it "Snacks," you can store data in a variable and give it a name.

In JavaScript:

  • You use let, const, or var to create a variable.
  • let is used to declare a variable that can be reassigned.
  • const is used to declare a constant variable that cannot be reassigned.
  • We do not use var to declare variables because it has function scope and can lead to unexpected behavior.

Example of using let in JavaScript:

let snack = "Chips";  // A box named "snack" with "Chips" inside
console.log(snack);   // Outputs: Chips
snack = "Cookies";    // Update the snack variable
console.log(snack);   // Output: Cookies

Example of using const in JavaScript:

const name = "John Doe";  // A box named "name" with "John Doe" inside
console.log(name) ;       // Outputs: John Doe
name = "Jane Doe";        // This will cause an error because name is a constant

Note:
const num;                // Cannot declare a constant variable without initialization

Understanding Data Types in JavaScript

In JavaScript, data types tell us what kind of value a variable can store. Think of it like different types of containers that can hold different kinds of items. For example, you can have a container for numbers, a container for words, or a container for true/false values. Here are the main data types:

  • String: Used for text, like "Hello, world!".
  • Number: Used for numbers, like 25 or 3.14.
  • Boolean: Can be either true or false, like yes or no.
  • Object: A container for related data, like a person's name, age, and address.
  • Array: A special type of object for storing a list of values, like [1, 2, 3, 4].
  • Null: Represents the absence of a value or nothing.
  • Undefined: A variable that has been declared but not assigned a value.

Examples:

let str = "Hello, world!";      // String data type
let num = 42;                   // Number data type
let rating = 3.14;              // Floating-point number (also a number data type)
let isTrue = true;              // Boolean data type
let isFalse = false;            // Boolean data type
let emptyValue = null;          // Null data type
let notDefined;                 // Undefined data type
let person = { 
    name: "John", 
    age: 30,  
    city: "New York"
};                              // Object data type
let numbers = [1, 2, 3, 4, 5];  // Array data type

The typeof operator is used to check the data type of a variable or value. It tells you what kind of value a variable is holding.

Examples of using typeof:

console.log(typeof str);        // Output: string
console.log(typeof num);        // Output: number
console.log(typeof rating);     // Output: number
console.log(typeof isTrue);     // Output: boolean
console.log(typeof isFalse);    // Output: boolean
console.log(typeof emptyValue); // Output: object (this is a known quirk in JavaScript)
console.log(typeof notDefined); // Output: undefined
console.log(typeof person);     // Output: object
console.log(typeof numbers);    // Output: object

Why both person and numbers are "object":

In JavaScript, both objects and arrays are technically types of objects. While arrays are a special kind of object, typeof will return "object" for both. To distinguish between an array and a regular object, you can use Array.isArray():

console.log(Array.isArray(numbers)); // Outputs: true
console.log(Array.isArray(person));  // Outputs: false
