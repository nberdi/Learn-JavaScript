What is JavaScript?

  • JavaScript is a programming language that adds interactivity to websites.
  • It makes web pages dynamic by responding to user actions like clicks and typing.
  • You can use JavaScript to create animations, forms, and interactive elements.
  • It runs in the browser, which means users don't need to install anything.
  • JavaScript works with HTML and CSS to build modern websites and web apps.
  • Popular websites use JavaScript to improve user experience and functionality.

Getting Started with JavaScript

  1. Create an HTML File: Open a text editor and save the file as index.html.
  2. Add Basic HTML: Add the basic structure with <!DOCTYPE html>, <html>, <head>, and <body>.
  3. Add a Script Tag: Inside the <body>, add a <script> tag.
  4. Insert JavaScript Code: JavaScript code is inserted between the <script> and </script> tags in HTML.
  5. Write Your First Script: Add a simple alert like alert('Hello, JavaScript!');.
  6. Save and Open: Save the file and open it in your browser.
  7. Explore the Browser Console: Right-click and select "Inspect" to open the console and test JavaScript.

Understanding Variables in JavaScript

Think of a variable like a labeled box where you can store information. Just like you put your favorite snacks in a box and label it "Snacks," you can store data in a variable and give it a name.

In JavaScript:

  • You use let, const, or var to create a variable.
  • let is used to declare a variable that can be reassigned.
  • const is used to declare a constant variable that cannot be reassigned.
  • We do not use var to declare variables because it has function scope and can lead to unexpected behavior.

Example of using let in JavaScript:

let snack = "Chips";  // A box named "snack" with "Chips" inside
console.log(snack);   // Outputs: Chips
snack = "Cookies";    // Update the snack variable
console.log(snack);   // Output: Cookies

Example of using const in JavaScript:

const name = "John Doe";  // A box named "name" with "John Doe" inside
console.log(name) ;       // Outputs: John Doe
name = "Jane Doe";        // This will cause an error because name is a constant

Note:
const num;                // Cannot declare a constant variable without initialization

Understanding Data Types in JavaScript

In JavaScript, data types tell us what kind of value a variable can store. Think of it like different types of containers that can hold different kinds of items. For example, you can have a container for numbers, a container for words, or a container for true/false values. Here are the main data types:

  • String: Used for text, like "Hello, world!".
  • Number: Used for numbers, like 25 or 3.14.
  • Boolean: Can be either true or false, like yes or no.
  • Object: A container for related data, like a person's name, age, and address.
  • Array: A special type of object for storing a list of values, like [1, 2, 3, 4].
  • Null: Represents the absence of a value or nothing.
  • Undefined: A variable that has been declared but not assigned a value.

Examples:

let str = "Hello, world!";      // String data type
let num = 42;                   // Number data type
let rating = 3.14;              // Floating-point number (also a number data type)
let isTrue = true;              // Boolean data type
let isFalse = false;            // Boolean data type
let emptyValue = null;          // Null data type
let notDefined;                 // Undefined data type
let person = { 
    name: "John", 
    age: 30,  
    city: "New York"
};                              // Object data type
let numbers = [1, 2, 3, 4, 5];  // Array data type

The typeof operator is used to check the data type of a variable or value. It tells you what kind of value a variable is holding.

Examples of using typeof:

console.log(typeof str);        // Output: string
console.log(typeof num);        // Output: number
console.log(typeof rating);     // Output: number
console.log(typeof isTrue);     // Output: boolean
console.log(typeof isFalse);    // Output: boolean
console.log(typeof emptyValue); // Output: object (this is a known quirk in JavaScript)
console.log(typeof notDefined); // Output: undefined
console.log(typeof person);     // Output: object
console.log(typeof numbers);    // Output: object

Why both person and numbers are "object":

In JavaScript, both objects and arrays are technically types of objects. While arrays are a special kind of object, typeof will return "object" for both. To distinguish between an array and a regular object, you can use Array.isArray():

console.log(Array.isArray(numbers)); // Outputs: true
console.log(Array.isArray(person));  // Outputs: false

String Concatenation and Template Literals

  • String Concatenation: We join strings using the + operator to combine values into a single string.

Example:

let firstName = "John";
let lastName = "Doe";
let fullName = firstName + " " + lastName; // Concatenate first and last name
console.log(fullName);                     // Output: John Doe
console.log("Full name is: " + fullName);  // Output: Full name is: John Doe
  • Template Literals: Introduced in ES6, these use backticks `...` and ${} to embed expressions (like variables) directly within strings, making the code cleaner and more readable.

Example:

let age = 30; 
let greeting = `Hello, my name is ${fullName} and I am ${age} years old.`; // Use template literals for string interpolation
console.log(greeting); // Output: Hello, my name is John Doe and I am 30 years old.

String Methods in JavaScript

Methods like toUpperCase(), toLowerCase(), replace(), split(), and others provide powerful ways to manipulate and interact with strings.

  • toUpperCase(): Converts a string to uppercase.
  • toLowerCase(): Converts a string to lowercase.
  • includes(): Checks if a string contains a specified substring.
  • startsWith(): Checks if a string starts with a specified substring.
  • endsWith(): Checks if a string ends with a specified substring.
  • indexOf(): Returns the index of the first occurrence of a substring.
  • replace(): Replaces a specified substring with another substring.
  • split(): Splits a string into an array based on a delimiter.
  • length: Returns the length of a string.
  • charAt(): Returns the character at a specified index.
  • charCodeAt(): Returns the ASCII code of the character at a specified index.
  • trim(): Trims whitespace from both ends of a string.
  • trimStart(): Trims whitespace from the start of a string.
  • trimEnd(): Trims whitespace from the end of a string.
  • repeat(): Repeats a string a specified number of times.
  • padStart(): Pads the start of a string with a specified character.
  • padEnd(): Pads the end of a string with a specified character.
  • substring(): Returns a part of a string between two indices.
  • slice(): Returns a portion of a string based on specified indices.

Examples:

let message = "Hello, World!";

// Convert to uppercase and lowercase
console.log(message.toUpperCase());                   // HELLO, WORLD!
console.log(message.toLowerCase());                   // hello, world!

// Check if message includes "World"
console.log(message.includes("World"));               // true

// Replace "World" with "JavaScript"
console.log(message.replace("World", "JavaScript"));  // Hello, JavaScript!

// Split message into array using ", " as delimiter
console.log(message.split(", "));                     // ["Hello", "World!"]

// Get the length of the message
console.log(message.length);                          // 13

// Trim the whitespace
console.log("  Hello  ".trim());                      // "Hello"

Advanced string operations:

// Using slice() vs substring()
let str = "Hello World";
console.log(str.slice(0, 5));                         // "Hello"
console.log(str.substring(0, 5));                     // "Hello"

// Split and join strings
let fruits = "apple,banana,orange";
let result = fruits.split(",");
console.log(result);                                  // ["apple", "banana", "orange"]

let greetingArr = ["Hello", "World"];
console.log(greetingArr.join(" "));                   // "Hello World"

// Reverse a string (split, reverse, join)
let word = "JavaScript";
let reversedWord = word.split("").reverse().join("");
console.log(reversedWord);                            // "tpircSavaJ"

Conditionals in JavaScript

If-Else Statement

The if-else statement is used to perform a block of code based on a condition.

Example:

const x = 10;
if (x > 5) {
    console.log("x is greater than 5!"); // Output: x is greater than 5!
} else if (x < 5) {
    console.log("x is less than 5!");
} else {
    console.log("x is equal to 5!");
}

Multiple Conditions

You can combine multiple conditions using logical operators like && (and) and || (or).

Example:

if (x > 5 && x < 15) {
    console.log("x is between 5 and 15!"); // Output: x is between 5 and 15!
} else if (x < 5 || x > 15) {
    console.log("x is either less than 5 or greater than 15!");
} else {
    console.log("x is exactly 5 or 15!");
}

Ternary Operator

The ternary operator is a shorthand way to write an if-else statement.

Example:

const result = (x > 5) ? "x is greater than 5!" : "x is not greater than 5!";
console.log(result); // Output: x is greater than 5!

Switch Statement

The switch statement is used to execute different code based on different cases.

Example:

switch (x) {
    case 5:
        console.log("x is equal to 5!");
        break;
    case 10:
        console.log("x is equal to 10!"); // Output: x is equal to 10!
        break;
    default:
        console.log("x is neither 5 nor 10!");
}
